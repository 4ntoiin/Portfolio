APT NEWS TRACKER

Threat Intelligence Python OSINT GitHub

// DESCRIPTION

Script Python qui collecte automatiquement les dernières actualités sur les groupes APT étatiques (APT28, Lazarus, APT41, Sandworm...) depuis des sources OSINT ouvertes. Génère un rapport HTML de veille quotidienne — exactement ce que fait un analyste Threat Intelligence au quotidien.

// FONCTIONNALITÉS

  • → Collecte RSS depuis blogs de sécurité (ANSSI, Mandiant, Kaspersky...)
  • → Filtrage par groupe APT et pays d'origine
  • → Rapport HTML dark mode généré automatiquement
  • → Mapping géopolitique des menaces détectées
  • → Envoi optionnel par email quotidien
